# elm学校课程设计项目
## 环境搭建
全局安装yarn
```
需要在管理员cmd中执行下面两条命令才能安装
npm cache clean --force
npm install -g --global-style yarn

```
## Project setup
```
yarn install//安装组件
```

### Compiles and hot-reloads for development
```
yarn serve//启动
```

### Compiles and minifies for production
```
yarn build
```
